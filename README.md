FridgeTrotter — це комплексна веб-платформа для мандрівників, яка поєднує в собі маркетплейс турів, соціальну мережу для пошуку попутників та унікальну гейміфіковану механіку колекціонування спогадів.
Головна "фішка" проєкту — віртуальний холодильник, де користувачі можуть збирати та розміщувати цифрові магніти з відвіданих країн та міст.

Основний функціонал:

Для мандрівників:
    - Пошук та бронювання турів: перегляд пропозицій від різних агенцій, фільтрація за категоріями (пляжний, гірський, міський туризм), ціною та рейтингом.
    - Пошук попутників: створення оголошень для пошуку компанії у подорож із зазначенням бюджету, дат та країни.
    - Форум: обговорення маршрутів, поради та обмін досвідом.
    - Віртуальний холодильник: інтерактивна дошка в профілі, куди можна додавати магніти, перетягувати їх та налаштовувати дизайн холодильника.
    - Чат: спілкування з іншими користувачами та агентами в реальному часі.

Для турагенцій та гідів:
    - Кабінет партнера: управління профілем агенції, додавання логотипу та опису.
    - Створення турів: публікація нових турів з фотографіями, програмою та доступними датами.
    - Управління бронюваннями: перегляд та зміна статусу заявок від клієнтів (підтвердження/відхилення).
    - Створення магнітів: можливість створювати унікальні магніти під конкретні тури або замовляти дизайн.
